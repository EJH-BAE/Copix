# Copix payment site (Netlify)

1. Edit Toss account details in `index.html` (TOSS object) and in `../src/services/paymentConfig.ts`.
2. Deploy this folder to Netlify (drag-drop or `netlify deploy --dir=payment-site`).
3. Set `PAYMENT_SITE_URL` in `../src/services/paymentConfig.ts` to your Netlify URL.
